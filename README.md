# AWS Native CI/CD Pipeline Project

## Project Overview

This project demonstrates Continuous Integration and Continuous Deployment using AWS native DevOps services.

## AWS Services Used

- GitHub
- AWS CodePipeline
- AWS CodeBuild
- AWS CodeDeploy
- Amazon EC2
- Amazon S3
- Amazon CloudWatch
